Platform

Four layers.
One platform.

Sentinel is not a dashboard bolted onto a sensor. It is a purpose-built pipeline — each layer engineered for its role, integrated end-to-end, deployed entirely within your infrastructure. No external services. No data leaving your perimeter.

Layer 01
Edge Agent
A lightweight binary that connects to your existing hardware — Modbus TCP, CAN Bus, MQTT, OPC-UA, Serial, Syslog, and more. No proprietary hardware required. Deploy once, connect anything. Fully configurable from the SHIP dashboard without touching the agent.
11 connectors
↓ normalized signal stream
Layer 02
SHIP Gateway
The secure transport and normalization layer. SHIP receives raw signal frames from the Edge Agent, validates and normalizes them into a unified evidence schema, enforces access control, and routes structured data to the analysis engine. Your single pane of glass for signal operations.
Real-time dashboard
↓ classified signal events
Layer 03
Hermes Engine
The deterministic analysis core. Hermes evaluates every signal event against 14 behavior classes — scoring severity, stability, timing, and anomaly intensity without probabilistic drift. Same input always produces the same output. Auditable, reproducible, defensible.
Deterministic · 14 classes
↓ evidence-grade incidents
Layer 04
Evidence Reporter
Every meaningful anomaly becomes a structured incident with full context — timestamps, signal chain, severity score, behavior classification, and decision record. Export as PDF. Archive for audit. 30-day on-premise retention with configurable policies.
PDF · Audit-ready
